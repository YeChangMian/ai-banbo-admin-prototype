# AI伴播产品原型

AI伴播是一款服务于内容电商直播的 AI 伴播内容生产与现场控制产品。商家可以使用数字模特、商品素材和展示模版，快速制作商品展示或直播互动视频，并在直播过程中按商品讲解、主播口令或评论触发播放。

## 面向用户

- 服饰、美妆、宠物等品类的品牌商家与抖店运营团队
- 负责直播选品、内容制作、直播中控和账号管理的业务人员
- 需要在高频直播中扩大商品展示覆盖、提升直播间互动的代运营团队

## 解决的问题

1. 主播无法在有限直播时间内完整展示商品的多款色、搭配效果和细节。
2. 伴播视频制作依赖专业人员，准备周期长，难以适配每日选品变化。
3. 直播过程中缺少能随商品讲解和互动节点即时出现的内容，运营只能依赖人工切换素材。

## 核心能力

- 从店铺商品中选择本次需要展示的款式，结合数字模特与展示模版制作伴播视频。
- 管理生成和上传的视频，配置商品讲解、主播口令、评论触发等播放条件。
- 在直播中控台通过浏览器插件查看播放队列、手动插播视频，并实时识别主播语音触发内容。
- 为团队成员提供店铺、直播账号和使用权限管理。

## 本地运行

在项目目录启动静态文件服务：

```bash
python3 -m http.server 8128
```

打开以下页面：

- 管理后台：`http://127.0.0.1:8128/index.html`
- 直播中控插件：`http://127.0.0.1:8128/live-console/index.html`

## 原型说明

- `index.html`：AI伴播管理后台原型
- `index-v3.html`：V3 简化版原型源文件
- `live-console/`：直播中控台浏览器插件原型
- `assets/`：商品与数字模特案例素材
- `PRD.md`：产品需求文档

当前为静态交互原型，不包含真实后端接口、抖音开放平台回调和 AI 视频生成服务。详细产品规则见 `PRD.md`。
